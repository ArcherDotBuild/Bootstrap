## 186. Spacing

### Spacing

- m: Margin
- p: Padding

### Values

- t: Top (mt-value or pt-value)
- b: Bottom (mb-value or pb-value)
- s: Start (ms-value or ps-value)
- e: End (me-value or pe-value) pe-none & pe-auto(Pointer events)
- x: Left and Right (mx-(-value) or px-(-value))
- y: Top and Bottom (my-(-value or mx-(-value))
- blank: all 4 sides (m-value or p-value)

### Sizes

- 0: Eliminates all values to zero
- 1: $spacer \* .25(1rem \* 0.25)
- 2: $spacer \* .5(1rem \* 0.5)
- 3: $spacer \* (1rem)
- 4: $spacer \* 1.5(1rem \* 1.5)
- 5: $spacer \* 3(1rem \* 3)
- auto

```
.mt-0 {
  margin-top: 0 !important;
}

.ms-1 {
  margin-left: ($spacer * .25) !important;
}

.px-2 {
  padding-left: ($spacer * .5) !important;
  padding-right: ($spacer * .5) !important;
}

.p-3 {
  padding: $spacer !important;
}
```
